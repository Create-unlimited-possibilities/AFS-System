## 功能定位

AFS AI陪伴功能是基于**LangChain生态系统**的升级版智能对话系统，核心特性：

- **动态角色卡加载**：基于用户A套问题生成个人画像，其他用户可通过专属编号与角色卡对话
- **预处理机制**：协助者的对话准则预先生成，对话时直接加载，提升性能
- **独立好感度系统**：每个对话者（协助者/陌生人）都有独立的好感度记录
- **LangGraph编排**：使用有状态的图状结构管理对话流程
- **增量更新**：协助者修改答案或新协助者加入时，仅更新受影响的部分

## 技术栈

| 层级 | 技术 | 说明 |
|------|------|------|
| 前端 | Next.js 15 + TypeScript + shadcn/ui | 现代化UI |
| 后端 | Node.js + Express + LangChain | API服务 |
| AI编排 | LangGraph | 有状态对话管理 |
| 向量检索 | langchain-chroma | ChromaDB集成 |
| 模型服务 | langchain-ollama + Ollama | LLM推理 |
| 数据库 | MongoDB | 结构化数据 |
| 文件系统 | JSON文件 | 向量索引、对话准则 |
| 部署 | Docker + Docker Compose | 容器化 |
