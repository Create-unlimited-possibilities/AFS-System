# AFS System

Welcome to **AFS System** (Ancestral Family Stories) - a digital memory preservation system designed for elderly users.

## Project Overview

AFS System is an innovative application that combines AI technology with family memory preservation. It helps elderly users record their life stories through guided conversations and preserves these precious memories for future generations.

## Core Values

- **Digital Memory Preservation** - Structured storage of life stories and precious memories
- **AI-Powered Companionship** - Personalized conversations using LangGraph technology
- **Family Collaboration** - Multi-user support for collaborative memory building
- **Privacy Protection** - Complete RBAC permission control system

## Key Features

| Feature | Description |
|---------|-------------|
| Memory Storage | Structured Q&A system with basic and emotional question levels |
| AI Companion | LangGraph-based personalized conversations with memory |
| Family Collaboration | Multiple family members can help answer questions |
| Role Card Generation | AI-generated character profiles with personality traits |
| RAG Retrieval | ChromaDB vector search for semantic memory retrieval |
| Permission Management | Complete RBAC system for data privacy |

## Technology Highlights

- **LangGraph** - Stateful conversation flow with memory persistence
- **RAG** - Retrieval-Augmented Generation with ChromaDB
- **Dual Storage** - MongoDB + File system for data reliability
- **Local AI** - Ollama-powered local embedding and LLM support
