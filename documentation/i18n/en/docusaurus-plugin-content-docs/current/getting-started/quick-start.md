# Quick Start

This guide will help you experience the core features of AFS System in 5 minutes.

## Prerequisites

Make sure you have completed the system deployment following the [Installation Guide](/docs/getting-started/installation).

Visit http://localhost:3002 to confirm the system is running.

## Step 1: Create an Account

### Register a New Account

1. Open your browser and visit http://localhost:3002
2. Click the "Register" button in the upper right corner
3. Fill in the registration information:
   - **Username**: Choose a memorable username
   - **Email**: Enter your email address
   - **Password**: Set a secure password
   - **Confirm Password**: Enter your password again
4. Click the "Register" button to complete registration

### First Login

1. After successful registration, the system will automatically redirect to the login page
2. Login using the email and password you registered with
3. Upon first login, the system will guide you to the welcome page

## Step 2: Complete Your Profile

### Fill in Basic Information

1. After logging in, click "Profile" in the left navigation bar
2. Fill in the following information:
   - **Nickname**: The name you want the AI companion to call you
   - **Age**: Helps the AI better understand your background
   - **Gender**: Select your gender
   - **Location**: The city where you currently live
3. Click the "Save" button

## Step 3: Answer Questions to Build Memories

### Start the Questionnaire

1. Click "Questionnaire" in the left navigation bar
2. The system will display a series of questions divided into two levels:
   - **Basic Layer Questions**: About daily life, family background, work experience
   - **Emotional Layer Questions**: About life insights, important relationships, values

### Answer Questions

1. Select a question and click to enter the details page
2. Type your answer in the text box
3. Answers can be a brief sentence or detailed memories
4. Click the "Submit" button to save your answer
5. Continue answering other questions

**Tip**: The more detailed your answers, the better the AI companion will understand you, and conversations will be more personalized.

### Invite Family to Help

1. On the questionnaire page, click "Invite Family"
2. The system will generate a 16-digit invite code
3. Share the invite code with your family or friends
4. They can use the invite code to register and help you answer questions

## Step 4: Start AI Conversations

### Enter the Chat Interface

1. Click "Chat" in the left navigation bar
2. The system will load the AI companion interface

### Send Your First Message

1. Type your greeting in the input box, for example:
   - "Hello, I'm a new user"
   - "Can you introduce yourself?"
   - "I'd like to talk about my past"
2. Click the send button or press Enter
3. The AI companion will provide personalized responses based on your memory profile

### Experience Personalized Conversations

AFS System's AI companion has the following characteristics:

- **Remembers who you are**: The AI understands your background based on your questionnaire answers
- **Understands your stories**: You can share more life experiences, and the AI will remember them
- **Familiar communication style**: The AI can imitate the tone of someone you know (like family) when conversing with you

## Step 5: Explore the Admin Panel (Administrators)

If you have administrator permissions, you can access the admin panel:

1. Click the user menu in the upper right corner
2. Select "Admin Panel"
3. In the admin panel, you can:
   - View system statistics
   - Manage all users
   - Edit questionnaire questions
   - View user memory data
   - Manage roles and permissions

## Common Features

### View My Memories

1. Click "Profile" in the left navigation bar
2. In the "My Memories" section, view all saved questionnaire answers

### Modify Answers

1. Go to the "Questionnaire" page
2. Find the question you want to modify
3. Click the question to enter the details page
4. Modify your answer and resubmit

### Export Memory Data

1. Go to the "Profile" page
2. Click the "Export Data" button
3. Select export format (JSON, CSV, etc.)
4. Download your memory data
